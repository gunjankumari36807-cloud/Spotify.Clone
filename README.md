# Spotify Clone

A modern web-based Spotify clone built with HTML, CSS, and JavaScript. This project replicates the core functionality and design of Spotify's web player.

## Features

- Responsive design that mimics Spotify's interface
- Working music player with play/pause functionality
- Progress bar with seek functionality
- Volume control with mute option
- Playlist navigation
- Keyboard shortcuts
- Beautiful animations and transitions

## Keyboard Shortcuts

- Space: Play/Pause
- Left Arrow: Previous track
- Right Arrow: Next track

## Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/spotify-clone.git
```

2. Open the project folder:
```bash
cd spotify-clone
```

3. Open `index.html` in your web browser

## Technologies Used

- HTML5
- CSS3 (with Grid and Flexbox)
- JavaScript (ES6+)
- Font Awesome Icons
